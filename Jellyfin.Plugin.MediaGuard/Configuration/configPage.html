MediarrGuard Settings

MediarrGuard detects corrupt or unplayable media files and automatically requests replacements from Sonarr (TV) and Radarr (Movies).

Sonarr (TV Shows)

Base URL for Sonarr (e.g. http://localhost:8989)
Found in Sonarr under Settings > General > API Key

Radarr (Movies)

Base URL for Radarr (e.g. http://localhost:7878)
Found in Radarr under Settings > General > API Key
